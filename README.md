To pull code:
1. install git
2. in a projects folder, run ```git clone https://github.com/playergraham/laserstrike.git```
3. laserstrike will be in [projects folder]/laserstrike

To run LOCALLY: 
1. install python
2. in terminal go to lasterstrike directory
3. run ```python -m http.server 5173```
4. open webbrowser to - ```http://127.0.0.1:5173```
